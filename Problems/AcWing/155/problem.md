# T155 内存分配
## 题意概要
请写一个程序，模拟内存分配。
传统内存分配过程：
1. 内存以内存单元为单位，每个单元有对应编号，从$0$到$n$。
2. 运行过程中，程序开始占用内存。对于每一个申请，有以下几点：
    - 开始时间$t$；
    - 内存量$m$；
    - 持续时间$p$。
在此期间，这部分RAM不能被其他进程篡改。
3. 申请流程：
    1. 有长度位$m$的地址片   
        - 直接分配该分区，有多个分区时取最小。
    2. 无分区
        Step 1.放入等待队列
        Step 2.当有足够分区大小时，立即取出队头，分配单元。
## 输入
格式：
``` in
N
T M P
...
T M P
0 0 0
```
## 输出
第一行是全部进程都运行完毕的时刻。
第二行是被放入过等待队列的进程总数。
