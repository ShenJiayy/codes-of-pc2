# C - FishBones
## 问题陈述

艺术家高砂创作了一件鱼骨架形状的物品。

该物体由 $N$ 根肋骨和一根脊柱组成。肋骨的编号为 $1$ 到 $N$ 。

他希望在 $N+1$ 根骨头上各写一条字符串，并满足以下所有条件。

- 写在脊柱上的字符串长度为 $N$ 。
- 对于每根肋骨 $i = 1, \dots, N$ ，以下条件成立。
    - 写在肋骨 $i$ 上的字符串长度为 $A_i$ 。
    - 写在肋骨 $i$ 上的字符串的 $B_i$ /th 字符等于写在脊柱上的字符串的 $i$ /th 字符。
- 写在 $N+1$ 块骨头上的每个字符串都是 $S_1, \cdots, S_M$ 中的一个（允许重复）。

$S_1, \cdots, S_M$ 是由小写英文字母组成的字符串，它们都是不同的。

请就每个 $j = 1, \cdots, M$ 回答下面的问题。

- 在满足条件的字符串的写法中，有没有一种写在书脊上的字符串是 $S_j$ ？

## 限制因素

- $N$ 是整数。
- $1 \leq N \leq 10$
- $A_i$ 和 $B_i$ 是整数。( $1 \leq i \leq N$ )
- $1 \leq B_i \leq A_i \leq 10$ ( $1 \leq i \leq N$ )
- $M$ 是整数。
- $1 \leq M \leq 200\,000$
- $S_j$ 是一个由小写英文字母组成的字符串。( $1 \leq j \leq M$ )
- $1 \leq |S_j| \leq 10$ ( $1 \leq j \leq M$ )
- $S_1, \cdots, S_M$ 是一对不同的字符串。

## Input

The input is given from Standard Input in the following format:

```
$N$
$A_1$ $B_1$
$\vdots$
$A_N$ $B_N$
$M$
$S_1$
$\vdots$
$S_M$
```

#### 输出

输出 $M$ 行。

如果存在写入满足条件的字符串的方法， $j$ \-行（ $1 \leq j \leq M$ ）应包含 `Yes`，否则应包含 `No`。
