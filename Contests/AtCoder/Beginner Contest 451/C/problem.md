#### 问题陈述

高桥正在管理花园里的树木数量。最初，花园里没有树。

依次给出 $Q$ 个查询。每个查询都是以下两种类型之一。处理完每个查询后，立即输出当前花园中树木的数量。

- `1 h` ：向花园中添加一棵高度为 $h$ 的新树。
- `2 h` ：移除花园中所有高度不超过 $h$ 的树木。

#### 限制因素

- $1 \le Q \le 3 \times 10^5$
- $1 \le h \le 10^9$
- 所有输入值均为整数。

#### 输入

输入内容由标准输入法提供，格式如下

```
$Q$
$\text{query}_1$
$\text{query}_2$
$\vdots$
$\text{query}_Q$
```

$\text{query}_i$ 代表 $i$ \-th 查询，是以下两种类型之一：

```
$1$ $h$
```
```
$2$ $h$
``` 

#### 输出

输出 $Q$ 行。

在处理完 $i$ \-th 查询后， $i$ \-行应包含花园中树木的数量。

### 样本输出 1

```
1
2
3
1
2
```

树的数量变化如下

- 增加一棵高度为 $5$ 的树。花园中有一棵高度为 $5$ 的树。
- 增加一棵高度为 $7$ 的树。花园中有两棵高度为 $5, 7$ 的树。
- 新增一棵高度为 $8$ 的树。花园中有三棵高度为 $5, 7, 8$ 的树。
- 移除高度不超过 $7$ 的树木。花园中有一棵高度为 $8$ 的树。
- 增加一棵高度为 $3$ 的树。花园中有两棵高度为 $8, 3$ 的树。
